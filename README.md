## Code-Quiz
 This is my 4th Week Challenge quiz application using HTML, CSS, and Javascript. This application emphasizes the use of Javascript to provide quiz questions and collect user data to determine whether the answers to a question are correct, this then generates a score and appends a final page of results from the user data.

## Bulit with
  * HTML
  * CSS
  * JS
  * JS DOM

 * Dev Tool
